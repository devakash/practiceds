To run the script please change the chrome driver location accordingly.
Scripts check for below 4 conditions.
a) if the page has title it will display "page title" else it will display "title doesn't exists."
b) if the page has errors it will display that page has PHP errors.
c) if the page has 404 error, it will show page has 404 error.
d) if the page has an image, it will check if image has an alternate text , if yes it will display.

To setup:
Extract the zip file and import the project as java project in Eclipse.
